File Inventory

Config is split into numbered files loaded in alphanumeric order via include ~/.config/sway/*-config in the main config entry point. System defaults are loaded first via include /etc/sway/config.d/*.

System Include (/etc/sway/config.d/*)

Two files are loaded from the system-level config directory before the user config:

FileWhat it doesRelevance
40-sway-background.conf Sets default wallpaper: output * bg /usr/share/backgrounds/sway/Sway_Wallpaper_Blue_1920x1080.png fill Overridden immediately by setsbg redisplay in 90-execs-config. Visible only for a split second on first boot.
50-systemd-user.conf exec systemctl --user import-environment DISPLAY WAYLAND_DISPLAY SWAYSOCK XDG_CURRENT_DESKTOP
exec dbus-update-activation-environment … XDG_CURRENT_DESKTOP=sway 		Important: sets XDG_CURRENT_DESKTOP=sway — apps check this for Wayland-specific behavior. The dbus-update-activation-environment call in 90-execs-config duplicates part of this but omits XDG_CURRENT_DESKTOP.

Verdict: The include /etc/sway/config.d/* line is needed for XDG_CURRENT_DESKTOP=sway propagation. The background file is harmless. Do not remove the include.
